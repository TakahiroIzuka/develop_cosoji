# コソージ　cosoji_product

* "すきま時間 × 清掃" わたしの10分を1000円に、スキマ時間でキレイなまちに


## 使い方

1. コソージサイトにアクセス https://cosoji.jp/worker/
2. スマホで仕事を選ぶ(日程・やりたいお仕事を探す)
3. スマホから応募する(内容を確認して、応募ボタンを押すだけ)
4. 作業する(スマホの案内に従うだけで作業完了)
5. 対価GET(スマホから簡単入力で報告すれば、報酬ゲット)


## 使用技術

* HTML/CSS
* PHP
* MySQL
* JavaScript/jQuery
